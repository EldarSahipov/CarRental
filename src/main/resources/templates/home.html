О проекте

Данное веб-приложение даст возможность пользователю оформить аренду автомобиля выбранной марки на оговоренный договором срок. Смотреть код можно на GitHub EldarSahipov.

Назначение разработки

Разработка данной программы велась в рамках курсового проекта по дисциплине «Безопасность систем баз данных». Веб–приложение предназначено для хранения информации об арендаторах и пользователях, сведений об автомобилях, сроках аренды. В системе имеется возможность оформления автомобиля через менеджера.

Поддержка

У сотрудников есть важная функция, которая позволяет присылать письма с ошибками и недоработками сервиса

Top Security

В данном проекте установлена последняя версия Spring Security. Не стоит забывать, что нет неуязвимых систем! Данные в базе данных не хранятся в открытом виде. Также для повышения защиты сотрудника приведены ряд мер безопасности Чтобы сотрудник начал пользоваться рабочим пространством, админ должен прислать письмо с логином и паролем.

Разграничение прав доступа

Данные о прокатах автомобилях и другая информация хранятся в базе данных. СУБД обеспечивает разграничение прав доступа. Админу предоставляется больше функционала, чем обычному пользователю
